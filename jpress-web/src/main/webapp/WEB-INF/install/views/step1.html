Welcome to JPress. Before starting, Jpress needs you to provide the following information.

1. Database name
2. Database user name
3. Database password
4. Database host

We will use this information to create a database information file called jboot.properties. If the automatic creation fails to succeed, don't worry, you can open it in the text editor jboot-sample.properties , Fill in the database information and save it separately as jboot.properties .

Most of the time, your website service provider will give you this information. If you have no information, you will need to contact them before continuing to install JPress.

If you are ready, then play happily ... If you need more help, you can visit Jpress official website Obtain.
